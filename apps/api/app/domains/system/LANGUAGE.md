# System Domain - Ubiquitous Language

## Core Entities

- **Setting**: Configuration parameters that control system behavior (e.g., "Allow Student Registration").
- **InstituteInfo**: Profile and branding information for the institution.
- **AuditLog**: A tamper-evident record of critical system actions and data changes.
- **FileMetadata**: Information about uploaded files, including storage paths and security tags.
- **ImportLog**: Records of bulk data import operations.

## Key Actions

- **Configure**: Updating system-wide parameters.
- **Audit**: Reviewing logs for security or compliance purposes.
- **Manage Files**: Handling the lifecycle of uploaded documents and media.
